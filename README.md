# Simple Comments Page

Aplicação simples, semelhante à interface de posts/comentários do Facebook criada utilizando Webpack, Babel, Webpack Dev Server e React.

## Installation

Clone the project with

```sh
git clone https://github.com/mailysantos/SimpleCommentsPage.git
```

Get in the path project, then install the dependencies with:

```sh
yarn
```

If you are in development environment, you can use the development server:

```sh
yarn dev
```
